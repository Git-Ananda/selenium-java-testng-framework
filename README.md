# Selenium Java TestNG Framework

A Selenium automation framework built using Java, TestNG, Maven, and the Page Object Model.

## Tech Stack

- Java 21
- Selenium WebDriver
- TestNG
- Maven
- Log4j2
- Extent Reports

## Design Patterns

- Page Object Model (POM)
- Singleton
- Factory Pattern
- ThreadLocal Driver Management

## Features

- Cross-browser support
- Configuration management
- Explicit waits
- Screenshot capture
- HTML reporting
- Logging
- Page Object Manager
